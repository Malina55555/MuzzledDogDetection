# MuzzledDogDetection
Веб-приложение со встроенным искусственным интеллектом для детекции собак без намордников в общественных местах.

### Последовательность установки и запуска:
-- склонировать репозиторий

-- скачать зависимости из "requirements.txt"

-- запустить файл "main.app"
